Proyecto: La Mosqueta (Juego de la Copa)
Este es un juego de simulación de "La Mosqueta" desarrollado en C++. A diferencia de un juego simple, este proyecto implementa un sistema de gestión de jugadores y lógica de apuestas persistente durante la ejecución.

Características principales
Gestión de Usuarios: Registro de jugadores con nombre, alias, edad y cédula.

Sistema de Apuestas: Los jugadores comienzan con un saldo inicial y pueden ganar o perder dinero según sus aciertos.

Estructuras de Datos: Uso avanzado de struct para organizar la información de los jugadores y el historial de apuestas.

Lógica de Juego: Incluye un algoritmo de azar para determinar la posición de la copa y un sistema de "validación de jugadas".

Cómo ejecutar
Descarga el código y abre el archivo proyectoCopa.zpr en ZinjaI.

Compila y ejecuta (F9).

Utiliza el menú principal para registrarte como usuario antes de comenzar a apostar.

Project: The Shell Game (La Mosqueta)
A C++ implementation of the classic "Shell Game" (locally known as La Mosqueta). This project goes beyond simple gameplay by incorporating a full player management system and betting mechanics.

Key Features
User Management: Register players with details such as name, alias, age, and ID.

Betting System: Players start with an initial balance and can place bets to increase their score or risk losing their funds.

Data Structures: Advanced use of structs to manage player profiles and betting history.

Game Logic: Implements randomized cup positioning and custom validation rules for each round.

How to Run
Download the repository and open proyectoCopa.zpr using the ZinjaI IDE.

Compile and run (F9).

Use the main menu to create a user profile before you start playing.
